Games and media have the power to captivate audiences on an incomparable scale - and this will never cease to amaze me.

As a kid I would dream up endless inventions and ideas. Yet I never had the resources nor the patience to put pen to paper. So instead I taught myself how to mock-up concepts and rapid prototypes with whatever I had at my disposal - i.e. Microsoft Office.

Now in the age of apps anything is possible. But old habits die hard - Office undeniably still makes the world go round. So here I've used PowerPoint and Excel as development platforms, and a little creativity, to showcase just what you can do...



Wave Defence


by T. Zhang and A. Wigley (2018)
for PowerPoint 2010 Desktop or Later

A side-on survival shooter inspired by Flash games of the 2000s, this game brings new meaning to the phrase "Death By PowerPoint".

Wave Defence utilises PowerPoint's native vector graphics for design, and Custom Animation for high-level What-You-See-Is-What-You-Get programming. Hyperlinks, Triggers, and Custom Shows complete the presentation's interactive functionality.

This game comes in two versions, Standard and Hard Mode. Hard Mode adds more weapons, and limited ammunition for anyone looking for a tougher challenge.



Neither version of the game requires any Visual Basic for Applications code to run. Any Macro-Enabled copies of these presentations may be malicious and should be deleted immediately.

Mine­sweeper


by T. Zhang (2017)
for Excel 2010 Desktop or Later

A take on the classic Windows game from a simpler time.

Minesweeper was originally included with Windows 3.1 to teach users how to Right-Click, but was removed in 2012 with the release of Windows 8. Now you can relive the nostalgia with the power of Excel!

The game is reverse-engineered from countless hours of playing Windows Minesweeper. Excel's inherent grid layout helps to visualise the board in development, with VBA Macros, Spreadsheet Functions, and Conditional Formatting holding the pieces together underneath.



This game requires VBA macros to be enabled. My code is not designed to be intentionally harmful, though I cannot guarantee the integrity of copies found elsewhere. Please feel welcome to review my code before enabling macros by pressing "Alt+F11" in Excel to reveal the editor.

2048


by T. Zhang (2016)
for Excel 2010 Desktop or Later

A port of the 2014 viral sensation by Gabriele Cirulli.

Tilt the tiles with the arrow keys, or "WASD". When equal tiles collide they merge - try to reach 2048 or beyond! Start a new game by pressing "N" or "Enter".

Programmed in a week during my internship to teach myself Visual Basic for Applications (VBA). I was pleasantly surprised by how well 2048 runs, considering my grossly unoptimised code.

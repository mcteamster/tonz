Games have always been an integral part of my life.

Beyond being a favourite pastime, they are a form of art. They've taught me lessons, invoked emotions, and opened doors of opportunity. I wouldn't be the same without playing, creating, and admiring games.

I first started making games with Microsoft Office as a kid. It's so more than just the tape that holds companies together. Look and you'll see what you can do with PowerPoint, Excel and a little creativity...



Halloween


by T. Zhang and A. Wigley (2018)
for Windows - PowerPoint 2010 or Later

A side-on survival shooter inspired by Flash games of the 2000s, this game brings new meaning to the phrase "Death By PowerPoint".

This game comes in two versions, Standard and Plus. Halloween+ adds more weapons, and limited ammunition for a tougher challenge.



Neither version of the game requires any Visual Basic for Applications code to run. Any Macro-Enabled copies of these presentations may be malicious and should not be opened.

Mine­sweeper


by T. Zhang (2017)
for Windows - Excel 2010 or Later

A take on the classic Windows game from a simpler time. Now you can relive the nostalgia with the power of Excel!

The game is reverse-engineered from countless hours of playing Windows Minesweeper. This was made possible using Excel's grid layout alongside VBA Macros and functions.



This game requires VBA macros to be enabled. Always exercise caution when running scripts from the internet.

2048


by T. Zhang (2016)
for Windows - Excel 2010 or Later

A port of the 2014 viral sensation by Gabriele Cirulli.

Tilt the tiles with the arrow keys, or "WASD". When equal tiles collide they merge - try to reach 2048 or beyond! Start a new game by pressing "N" or "Enter".

Programmed in a week during my internship to teach myself Visual Basic for Applications (VBA).



This game requires VBA macros to be enabled. Always exercise caution when running scripts from the internet.

Avalon


by T. Zhang (2018)
for Web - Mobile/Tablet/Desktop

A digital board and card system for The Resistance: Avalon by Don Eskridge.

A social deception game inspired by Arthurian Legend. Now with hotseat style pass-and-play!

Secretly assign roles, choose your digital board, and let the quests begin!



This web app is intended as a supplement, and is not a full featured replacement for the original board game.
